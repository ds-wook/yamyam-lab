# 맛집 분석 대시보드

이 프로젝트는 맛집 데이터를 분석하고 시각화하는 Streamlit 기반의 웹 대시보드입니다.

## 개요

이 대시보드는 세 가지 주요 분석 기능을 제공합니다:
1. 리뷰어 분석: 개별 리뷰어의 리뷰 패턴과 선호도 분석
2. 식당 분석: 개별 식당의 상세 정보와 리뷰 트렌드 분석
3. 카테고리 분석: 식당 카테고리별 통계와 트렌드 분석
4. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run apps/main.py
```

## 기능 설명

### 리뷰어 분석
- 리뷰어 ID 기반 검색
- 리뷰 통계 분석
- 선호 카테고리 분석
- 시간별 리뷰 패턴 분석

### 식당 분석
- 식당명 또는 카테고리 기반 검색
- 기본 정보 및 메뉴 정보 표시
- 리뷰 트렌드 분석
- 위치 정보 표시

### 카테고리 분석
- 계층적 카테고리 분석
- 카테고리별 통계
- 트렌드 분석
